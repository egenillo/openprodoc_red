  Username: root
  Password: admin (the password set during installation)

  kubectl get pods -n {{ .Release.Namespace }} -l app.kubernetes.io/instance={{ .Release.Name }}

4. View logs:
  Core Engine: kubectl logs -n {{ .Release.Namespace }} -l app.kubernetes.io/component=core-engine
